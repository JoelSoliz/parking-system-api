# Parking System

Parking System backend API.

# Prerequisites

- Python 3.8+
- make
- Docker
- Docker Compose

# Development

Set up your environment

```bash
make setup
```

Build the containers

```bash
make build
```

Start the service

```bash
make start
```

Run the tests

```bash
make test
```

Stop and remove containers, networks, and pyc

```bash
make clean
```
